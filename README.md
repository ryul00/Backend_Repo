## 📝 서비스 소개
기존 2D 웹 게임에서 두뇌 집중력·기억력·순발력 등 강화 목적의 복합 기능을 갖춘 게임이 부족하다는 문제 인식 </br>  
오락성 위주의 단순 게임에서 벗어나, 학습·집중력 향상 요소를 결합한 종합 두뇌 게임 “똑똑하ZOO” 기획  </br> 
싱글 플레이뿐 아니라, 실시간 소켓 통신을 활용한 2인 멀티플레이까지 확장하여 사용자 몰입도 강화
